# Food Delivery App

This repository contains a full-stack food delivery application.

## Structure

- `backend/` - Node.js backend API and database utilities
- `foodie-feed/` - Frontend application built with Vite, React, and TypeScript

## Usage

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd foodie-feed
npm install
npm run dev
```

## Notes

- The backend and frontend are separate projects within the same repository.
- The frontend repo is linked to `https://github.com/ParimalShindeAmpcus/foodie-feed.git` as a nested git repository.
